# 💰 Finance Manager - Python CLI

A simple personal finance manager built in Python.  
This project helps users track income and expenses, organize financial data, and analyze their budget directly through a command-line interface.

---

## 📌 Features

- Add income and expenses
- Edit and delete records
- View transactions in table format
- Categorize expenses (Food, Transport, Housing, etc.)
- Track financial data in real time

---

## 📂 Categories

- Food
- Transport
- Housing
- Health
- Education
- Leisure
- Other

---

## 📊 Planned Features

- Monthly budget planning
- Financial reports
- Expense analysis by category
- Monthly comparison
- Savings goals
- Alerts for overspending

---

## 💾 Data Storage

Currently uses Python lists (in-memory).  
Future version will support:

- CSV storage
- SQLite database
